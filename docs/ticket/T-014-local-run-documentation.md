# T-014 — Local run documentation

## Goal
Make it easy to install and run TEAmate MVP locally.

## Requirements / tasks
- Add `README.md` (root or `docs/`) with:
  - prerequisites (Python version, Ollama optional)
  - install dependencies
  - configure env vars
  - run the app
  - run tests
- Add troubleshooting section:
  - Ollama not running
  - model not pulled
  - slow embeddings

## Acceptance criteria
- A new developer can run the MVP from scratch following docs.
